liperpay
LiperPay is a next-generation payments infrastructure designed to power fast, secure, and scalable money movement across Africa and beyond. It provides the core financial rails for digital wallets, agent-based cash-in/cash-out, online payments, and future regulated financial services.

Built with scale, reliability, and real-world use cases in mind, LiperPay enables individuals, businesses, and platforms to move money efficiently through modern APIs and infrastructure.
